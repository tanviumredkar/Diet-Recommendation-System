# 🥗 Diet Recommendation System

NutriGuide is a diet planning system designed using HTML5 and CSS3 to provide structured meal recommendations based on user inputs and goals. The system helps promote healthier lifestyles by displaying customized diet plans for weight loss, weight gain, and maintenance.

## 🎯 Objectives
- To develop a smart diet recommendation system using Artificial Intelligence
- To provide personalized meal plans based on user inputs
- To support different health goals like weight loss, weight gain, and maintenance
- To improve overall health awareness and nutrition planning
  
## 🧠 Domain
Web Development (Frontend)

## ⚙️ Technologies Used

- HTML5
- CSS3

## 📊 Features
- Personalized diet recommendations
- Supports Veg and Non-Veg diet plans
- User-friendly interface
- Goal-based meal planning (Weight Loss, Gain, Maintain)
- Clean and responsive design
  
## 🛠️ System Architecture
- User enters details (age, weight, height, goal, etc.)
- Inputs are processed using predefined logic
- Diet plan is selected based on conditions
- Customized diet plan is displayed on screen

## 🚀 How to Run the Project
- Download or clone the repository
- Open the project folder
- Open the index.html file in any web browser

## 📷 Output Screens
- User Input Form
- Diet Plan Result Page
- Personalized Recommendations
