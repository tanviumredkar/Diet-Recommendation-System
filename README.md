🥗 NutriGuide: Personalized AI Diet Planner
📌 Overview
NutriGuide is an AI-powered diet planning system designed to provide personalized meal recommendations based on user health data, preferences, and goals. The system aims to promote healthier lifestyles by generating customized diet plans for weight loss, weight gain, and maintenance.
🎯 Objectives
To develop a smart diet recommendation system using Artificial Intelligence
To provide personalized meal plans based on user inputs
To support different health goals like weight loss, weight gain, and maintenance
To improve overall health awareness and nutrition planning
🧠 Domain
Artificial Intelligence in Healthcare
⚙️ Technologies Used
Programming Language: Python
Frontend: HTML, CSS
Backend: Flask / Django (if applicable)
Machine Learning Algorithm: Random Forest Classifier
Database: MySQL / SQLite
📊 Features
Personalized diet recommendations
Supports Veg and Non-Veg diet plans
User-friendly interface
Goal-based meal planning (Weight Loss, Gain, Maintain)
Quick and efficient prediction using ML model
🛠️ System Architecture
User enters details (age, weight, height, goal, etc.)
Data is processed and analyzed
Machine Learning model predicts suitable diet
Customized diet plan is generated and displayed
